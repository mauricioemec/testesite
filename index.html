Ajuda

Início Rápido

  1. Comece cadastrando seus imóveis na seção "Imóveis"
  2. Se houver financiamentos ou consórcios, registre-os nas respectivas seções
  3. Registre reformas realizadas em cada imóvel
  4. Lance receitas de hospedagem conforme as reservas
  5. Registre todas as despesas operacionais
  6. Acompanhe os resultados no Dashboard e relatórios financeiros

Glossário

ADR (Average Daily Rate)
Diária média - receita por diária vendida
RevPAR (Revenue per Available Room)
Receita por quarto disponível - ADR × Taxa de Ocupação
NOI (Net Operating Income)
Receita operacional líquida antes de despesas financeiras
LTV (Loan to Value)
Relação entre dívida e valor do imóvel (Dívida / Valor de Mercado)
EBITDA
Lucro antes de juros, impostos, depreciação e amortização
Cap Rate
Taxa de capitalização - NOI / Valor do Imóvel
DCF (Discounted Cash Flow)
Fluxo de caixa descontado - método de valuation
NAV (Net Asset Value)
Valor patrimonial líquido
Equity
Patrimônio líquido no imóvel (Valor - Dívida)

FAQ

Como os dados são salvos?

Todos os dados são salvos automaticamente no navegador (localStorage). Para fazer backup, use o botão "Exportar dados".

Posso usar em múltiplos dispositivos?

Os dados ficam salvos apenas no navegador atual. Para sincronizar, exporte os dados de um dispositivo e importe no outro.

Como funciona a depreciação?

A depreciação é calculada linearmente com base na vida útil estimada de cada ativo.

Qual a diferença entre SAC e PRICE?

SAC: amortização constante, parcelas decrescentes. PRICE: parcelas fixas, amortização crescente.
